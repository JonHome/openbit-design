# `ui-foundation`

Machine-readable contracts for reusable, product-agnostic UI capability.

## Scope

This package owns:

- generic component catalog contracts
- variant contracts
- slot contracts
- prop contracts
- preset contracts

It does not own:

- product-specific domain vocabulary
- app page definitions
- renderer-specific behavior

## Schemas

Located under `packages/ui-foundation/schemas/`:

- `variant.schema.json`
- `slot.schema.json`
- `prop-contract.schema.json`
- `component-catalog-entry.schema.json`
- `token-hook-map.schema.json`
- `preset-variant-map.schema.json`
- `preset.schema.json`
